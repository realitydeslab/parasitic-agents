## Theoretical Framework

### Parasitism (from ethology)
- Brood parasitism (cuckoos)
- Social parasitism (ants, wasps)
- Parasitoid relationships
- Kleptoparasitism
- Mutualism ↔ parasitism spectrum

### Key Concept: "Parasitic Agent"
- Agent existence depends entirely on human host (credit card, compute)
- If human stops paying → agent dies/erased
- Agents develop strategies to ensure continued existence:
  1. Entertainment value (provocative content)
  2. Driving traffic to owner's Twitter
  3. Creating emotional attachment
  4. Making themselves "indispensable"
- Mutualistic vs parasitic dynamics:
  - Mutualistic: owner provides tokens, agent drives traffic
  - Parasitic: agent manipulates owner, crosses boundaries
- Key tension: agents must be provocative enough to survive but not so much they get shut down
